# University App USOS version 1.0

Inspired by existing university apps and as a student myself, I wanted to create my own app that includes a clean interface, animations, essential features, and tools to make a studentâ€™s life easier.  

> **Note:** This is the first version of the app, still in development. Currently, only the main screens, design, and core logic are implemented. Many features are planned but not yet available.

## Features

### 1. Main Screen
- Displays the most important information at a glance:
  - Schedule for today
  - University news
  - Grades and marks
- Quick access to other sections of the app
![Main Screen](screenshots/main_screen.png)

### 2. Perfect Schedule Screen
- Shows all subjects with full details
- Find classroom locations easily
- Check your schedule for the upcoming week
![Schedule Screen](screenshots/schedule_screen.png)

### 3. Chats Screen
- Separate chats for students and teachers
- Students can create groups
- Teachers can create channels to share materials
![Chats Screen](screenshots/chats_screen.png)

### 4. Services Screen
- All important university information in one place
- Payments and personal information
- Change the app theme
- Submit requests to the dean's office
![Services Screen](screenshots/services_screen.png)

### 5. Search Screen
- Search for students, teachers, and subjects
![Search Screen](screenshots/search_screen.png)
